# web-scraping-csv-generator

## Overview

Welcome to the **web-scraping-csv-generator** project! This Python-based web scraping endeavor focuses on extracting valuable data from [Universidad Peru](https://www.universidadperu.com/), specifically targeting information related to companies engaged in the cultivation of fruits and vegetables. The project utilizes Selenium and BeautifulSoup to provide a structured approach to web scraping, data cleaning, and organization.

## Key Features

- **Selenium and BeautifulSoup:** Employs industry-standard libraries for effective web scraping.
- **Data Extraction and Cleaning:** Implements a comprehensive process for extracting, cleaning, and generating CSV files.
- **Recursive Scraping:** Utilizes a recursive scraping technique for multi-level data retrieval.

## Usage

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Rahuljangs/web-scraping-csv-generator.git
   ```

2. **Install Dependencies:**
   ```bash
   pip install bs4 selenium playsound
   ```

3. **Web Scraping:**
   - Execute the primary Python notebook: `main_scraping_notebook.ipynb`

4. **Data Cleaning:**
   - Utilize the cleaning notebook: `data_cleaning_and_csv_generation.ipynb`
